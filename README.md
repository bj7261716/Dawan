# 大灣藥局 - 後台管理系統

## 🚀 快速開始

### 方法1：直接開啟（最簡單）
1. 打開 `admin` 資料夾
2. 雙擊 `index.html` 檔案
3. 瀏覽器會自動開啟網站

### 方法2：使用本地伺服器（推薦）
```bash
cd admin
python -m http.server 8000
```
然後訪問：http://localhost:8000
