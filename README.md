## 🚀 快速開始

### 一鍵啟動（推薦）⭐

#### Windows
```bash
雙擊執行 start.bat
```

#### macOS / Linux
```bash
chmod +x start.sh
./start.sh
```

### 訪問系統
```
網址：http://localhost:5000
帳號：kuma
密碼：kuma!2022
```
