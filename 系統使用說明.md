## 🎯 系統簡介

大灣藥局後台管理系統是一個完整的電商後台管理平台，提供商品、訂單、會員、內容等全方位管理功能。

### 技術棧
- **前端**：Bootstrap 5, jQuery, Morris.js
- **後端**：Python Flask
- **資料庫**：SQLite
- **圖表**：Morris.js (基於Raphael.js)

### 系統特色
✅ 完整的CRUD功能  
✅ 即時數據統計圖表  
✅ 響應式設計  
✅ RESTful API設計  
✅ 本地運行，無需外部伺服器

## 🚀 快速開始

### 方法1：使用啟動腳本（推薦）

#### Windows
```bash
# 雙擊執行
start.bat

# 或在命令列執行
./start.bat
```

#### macOS / Linux
```bash
# 添加執行權限
chmod +x start.sh

# 執行腳本
./start.sh
```

### 方法2：手動啟動

#### 步驟1：建立虛擬環境
```bash
python -m venv venv
```

#### 步驟2：啟動虛擬環境

**Windows:**
```bash
venv\Scripts\activate
```

**macOS/Linux:**
```bash
source venv/bin/activate
```

#### 步驟3：安裝依賴
```bash
pip install -r requirements.txt
```

#### 步驟4：初始化資料庫（首次使用）
```bash
cd backend
python -c "from app import init_db; init_db()"
python seed_data.py
```

#### 步驟5：啟動服務器
```bash
python app.py
```

### 訪問系統
```
網址：http://localhost:5000
帳號：kuma
密碼：kuma!2022
```

## 📡 API文檔

### 基礎URL
```
http://localhost:5000/api
```

### 認證
所有API請求需要先登入，系統使用Session進行認證。

### 通用響應格式
```json
{
    "success": true|false,
    "message": "訊息內容",
    "data": { ... }
}
```

---

### 儀表板 API

#### GET /api/dashboard/stats
獲取儀表板統計數據

**響應示例**：
```json
{
    "success": true,
    "data": {
        "order_count": 50,
        "product_count": 10,
        "contact_count": 0,
        "view_count": 600,
        "latest_orders": [...],
        "low_stock_products": [...]
    }
}
```

#### GET /api/dashboard/order-chart
獲取訂單統計圖表數據（按月）

**響應示例**：
```json
{
    "success": true,
    "data": [
        {"month": "1月", "count": 5},
        {"month": "2月", "count": 8},
        ...
    ]
}
```

#### GET /api/dashboard/view-chart
獲取瀏覽統計圖表數據（按月）

**響應示例**：
```json
{
    "success": true,
    "data": [
        {"month": "1月", "count": 120},
        {"month": "2月", "count": 150},
        ...
    ]
}
```

---

### 商品管理 API

#### GET /api/products
獲取商品列表

**查詢參數**：
- `page`: 頁碼（預設：1）
- `per_page`: 每頁數量（預設：20）

**響應示例**：
```json
{
    "success": true,
    "data": {
        "products": [...],
        "total": 10,
        "page": 1,
        "per_page": 20
    }
}
```

#### GET /api/products/{id}
獲取商品詳情

**響應示例**：
```json
{
    "success": true,
    "data": {
        "id": 1,
        "name": "ADR-1益生菌膠囊",
        "sku": "ADR001",
        "price": 1280,
        ...
    }
}
```

#### POST /api/products
新增商品

**請求body**：
```json
{
    "category_id": 1,
    "name": "商品名稱",
    "sku": "SKU001",
    "description": "商品描述",
    "price": 1000,
    "original_price": 1200,
    "stock": 100,
    "unit": "個",
    "image": "/images/product.jpg",
    "status": 1
}
```

#### PUT /api/products/{id}
更新商品

**請求body**：同新增商品

#### DELETE /api/products/{id}
刪除商品

---

### 訂單管理 API

#### GET /api/orders
獲取訂單列表

**查詢參數**：
- `page`: 頁碼（預設：1）
- `per_page`: 每頁數量（預設：20）

#### GET /api/orders/{id}
獲取訂單詳情

**響應示例**：
```json
{
    "success": true,
    "data": {
        "order": {
            "id": 1,
            "order_no": "2026010212345",
            "status": "processing",
            ...
        },
        "items": [
            {
                "product_name": "ADR-1益生菌膠囊",
                "quantity": 2,
                "price": 1280,
                ...
            }
        ]
    }
}
```

#### PUT /api/orders/{id}/status
更新訂單狀態

**請求body**：
```json
{
    "status": "shipping"
}
```

**訂單狀態**：
- `pending`: 待處理
- `processing`: 處理中
- `shipping`: 配送中
- `completed`: 已完成
- `cancelled`: 已取消

## ❓ 常見問題

### Q1: 啟動時出現 "找不到 Python" 錯誤
**A**: 請確認已安裝 Python 3.8 或以上版本，並加入系統環境變數。

下載：https://www.python.org/downloads/

### Q2: 啟動時出現 "No module named 'flask'" 錯誤
**A**: 請確認已啟動虛擬環境並安裝依賴：
```bash
venv\Scripts\activate
pip install -r requirements.txt
```

### Q3: 無法訪問 http://localhost:5000
**A**: 
1. 檢查Flask是否正常啟動（終端機應顯示 "Running on http://0.0.0.0:5000"）
2. 檢查5000端口是否被佔用
3. 嘗試訪問 http://127.0.0.1:5000

### Q4: 圖表無法顯示
**A**: 
1. 檢查瀏覽器Console是否有JavaScript錯誤
2. 確認資料庫中有數據（執行 seed_data.py）
3. 確認Flask服務器正在運行

### Q5: 登入後一直跳回登入頁
**A**: 
1. 檢查瀏覽器Cookie設定（需允許Cookie）
2. 清除瀏覽器快取和Cookie後重試
3. 確認Flask的 SECRET_KEY 設定正確

### Q6: 如何重置資料庫
**A**: 
```bash
cd backend
rm dawan.db
python -c "from app import init_db; init_db()"
python seed_data.py
```

### Q7: 如何修改預設帳號密碼
**A**: 修改 `backend/seed_data.py` 中的管理員資料，然後重新執行：
```bash
python seed_data.py
```

### Q8: 如何在其他電腦訪問
**A**: 將Flask啟動時的 `host='0.0.0.0'` 保持不變，然後在其他電腦訪問：
```
http://[你的IP地址]:5000
```

## 🔧 進階設定

### 修改端口
編輯 `backend/app.py` 最後一行：
```python
app.run(debug=True, host='0.0.0.0', port=8080)  # 改為8080端口
```

### 關閉Debug模式（生產環境）
```python
app.run(debug=False, host='0.0.0.0', port=5000)
```

### 設定SECRET_KEY
編輯 `backend/app.py`：
```python
app.config['SECRET_KEY'] = '你的安全密鑰'
```
