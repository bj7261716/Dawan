## 🛠️ 下載方式

### 方法1：使用Chrome開發者工具（推薦，最可靠）

1. **打開原網站**：https://info.ouorange.com/kuma.com/admin
2. **登入帳號**：kuma / kuma!2022
3. **開啟DevTools**：按 F12
4. **切換到Network標籤**
5. **篩選資源類型**：
   - 點擊 "Img" 查看圖片
   - 點擊 "Font" 查看字體
   - 點擊 "All" 查看所有資源
6. **重新整理頁面**：按 F5
7. **下載資源**：
   - 找到你要的檔案
   - 右鍵 → "Open in new tab"
   - 另存檔案到對應的目錄

### 方法2：使用curl命令（批次下載）

我可以幫你寫一個批次下載腳本，一次下載所有資源。

### 方法3：使用wget（Linux/Mac）

```bash
wget https://info.ouorange.com/kuma.com/admin_assets/fonts/Material-Design-Iconic-Font.woff2
```

## ⚙️ 如何判斷是否缺失字體檔案？

### 檢查方式1：瀏覽器Console
1. 開啟 `admin/index.html`
2. 按 F12 → Console標籤
3. 查看是否有 404 錯誤（找不到檔案）
4. 特別注意 `.woff2`, `.woff`, `.ttf`, `.eot` 結尾的錯誤

### 檢查方式2：Network標籤
1. F12 → Network標籤
2. 重新整理頁面
3. 篩選 "Font"
4. 查看是否有紅色的失敗請求

### 檢查方式3：視覺檢查
如果圖標顯示為：
- 空白方框 □
- 問號 �
- 亂碼

代表字體檔案缺失或路徑錯誤。
